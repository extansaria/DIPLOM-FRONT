# Frontend (React + JavaScript)

Фронтенд реализован без TypeScript и без отдельных CSS-файлов.

## Особенность по стилям

Все стили находятся в корневом `index.html` в теге `<style>`.

## Структура

- `src/components` — переиспользуемые UI-компоненты.
- `src/pages` — основные разделы сайта.
- `src/data` — mock-данные.
- `src/utils` — вспомогательные функции.

## Основная логика

- Личный кабинет открывается в модальном окне по иконке справа сверху.
- Карточка упражнения открывается модально поверх каталога.
- Главная содержит hero, поиск и интерактивную анатомическую схему (вид спереди/сзади).
